### 中英文说明（点击按钮切换）

GitHub 仓库里的 `README.md` **不能运行脚本**，无法在页面内做「只显示一种语言、点击切换」。  
因此完整说明放在站点上的独立页面（部署后与博客一起发布）：

**→ [readme-bilingual.html](https://erdemtbao.github.io/blog/readme-bilingual.html)**（**English** / **中文** 按钮切换，并会记住你的选择）

本地预览：运行 `pnpm dev` 后打开  
`http://localhost:4321/blog/readme-bilingual.html`（与线上 `base: /blog` 一致）  
也可直接用浏览器打开仓库里的 [`public/readme-bilingual.html`](public/readme-bilingual.html)。
